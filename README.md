# 🚗 Campus Ride Share

A full-stack ride-sharing web application for university students to share rides, split costs, and travel together safely.

## 📌 Overview

Campus Ride Share helps students coordinate shared travel to common destinations like airports, railway stations, and city centers. It reduces transportation costs, promotes eco-friendly travel, and builds community among students.

## ✨ Features

### 👤 User Features
- **Secure Registration & Login** – JWT-based authentication with bcrypt password hashing
- **Post a Ride** – Share travel plans with destination, date, time, seats, and cost
- **Search & Find Rides** – Filter by destination and date
- **Join & Leave Rides** – Automatic seat availability management
- **Smart Cost Splitting** – Per-person cost calculated automatically
- **Personal Dashboard** – View posted rides and joined rides
- **Cancel Rides** – Drivers can cancel their own rides

### 👑 Admin Features
- View all registered users
- Delete inappropriate users/rides
- System statistics dashboard

### 🎨 UI/UX
- Clean red & white university-themed design
- Fully responsive (mobile, tablet, desktop)
- Loading indicators and error handling
